
# Quick Start Guide

## For Windows Users

### First Time Setup

1. **Open Terminal** (Command Prompt or PowerShell):
   ```bash
   cd C:\path\to\legal_assistant_desktop
   ```

2. **Install dependencies** (one time):
   ```bash
   yarn install
   ```

3. **Run the app:**
   ```bash
   yarn dev
   ```

The app will open in its own window!

---

## Daily Use

**To start the app:**
```bash
cd C:\path\to\legal_assistant_desktop
yarn dev
```

**To make changes:**
1. Edit any file in `components/` or `app/`
2. Save
3. App reloads automatically

**To create installer:**
```bash
yarn package
```
Find `Legal Assistant Setup.exe` in `release/` folder
